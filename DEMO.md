## What is Marp?

Marp is a Markdown presentation writer for developers. It is a tool to create slide decks from Markdown files.

---

## Why Marp?

- It is easy to use
- It is free
- It is open source
- It is cross-platform
- It is fast
- It is customizable
- It is extensible

---

## How to use Marp?

- Install Marp
- Create a Markdown file
- Run Marp
